== Description ==
This is the Web Stories Enhancer Plugin for showing the web stories to the website with the help of shortcode [web_stories_enhancer].
It shows Instagram-style latest stories in the round circle format which outputs with the help of Shortcode. It can be literally everywhere. 

<strong> Compatible Plugins  </strong> <br>
* Web Stories by Google (**[view](https://wordpress.org/plugins/web-stories/)** )
* MakeStories (for Web Stories) by MakeStories (**[view](https://wordpress.org/plugins/makestories-helper/)** )
* We plan to add many other plugin compatibilities in the future.

If you have any feature requests, please add them to the forums section and we will add them immediately. 

We are actively developing this plugin and also providing technical support as well.

== Features ==

* New - Advertisment Inserter for Webstories
* New - Dark mode compatible
* Web Stories will be displayed in Instagram style
* Display Web Stories anywhere using just a shortcode

== Frequently Asked Questions ==

= Do I need to install Web Stories by Google or MakeStories (for Web Stories) by MakeStories? =

Yes, this plugin requires one of the above plugin to work.

= How this plugin works? =

You just have to add the shortcode [web_stories_enhancer] and it will work out of the box.

= Do this work on my already existing web stories? =

Yes , this plugin work on stories made by  Web Stories by Google and MakeStories (for Web Stories) by MakeStories.

= Can I add WebStories any page / post ? =

Yes, you can. Just use this shortcode [web_stories_enhancer] 

= Can I get support regarding this plugin? =

Yes, you easily contact us by submitting the support form in plugin setting page
 

== Installation ==
1. Upload \"web-stories-enhancer.zip\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Add the shortcode \"[web_stories_enhancer]\" inside content or sidebar.
